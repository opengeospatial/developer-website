GeoPackage
==========

Prerequisites
-------------
The following are prerequisites for this training course:

* Familiarity with SELECT and CREATE commands in SQLite. A `tutorial <https://www.w3schools.com/sql/>`_ is available online.
* One of the sections requires access to SQLite3, the command line tool that comes with SQLite. The same section also requires access to a GeoPackage viewer such as QGIS.
