Web Feature Service (WFS)
========================

Prerequisites
-------------
The following are prerequisites for this section of the training:

* Knowledge of how the internet and the World Wide Web operate from a server/client standpoint
* Familiarity with Extensible Markup Language (XML). A `tutorial <https://www.w3schools.com/xml/>`_ on XML is available online.
