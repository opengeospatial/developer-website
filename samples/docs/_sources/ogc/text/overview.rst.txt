Overview
========

Audience
--------
Trainees that may need to interact with different groups and committees within the OGC.

Learning Objectives
-------------------
At completion of the module trainees will be able to:

- Explain what OGC is
- Explain what the OGC Technical Committee, Domain Working Groups and Standards Working Groups are.
- Define what a standard is
