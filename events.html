  • 2022-06-21

    Vector Data Code Sprint

    The OGC invites software developers to the July 2022 Vector Data Code Sprint, to be held from July 12-14, 2022, with a pre-event webinar on June 30. The Sprint will focus on OGC API - Features, OGC API - Moving Features, OGC API - Routes and OGC API - GeoVolumes.

  • 2022-04-12

    Space Partitions Code Sprint

    The OGC invites software developers to the May 2022 Space Partitions Code Sprint, to be held from May 10-12, 2022, with a pre-event webinar on May 4. The Sprint will focus on OGC API - Discrete Global Systems (DGGS), OGC API - Tiles, OGC API - Coverages, and OGC API - Environmental Data Retrieval (EDR).

    Read more>>
  • 2022-01-21

    2022 Joint OGC OSGeo ASF Code Sprint

    The Open Geospatial Consortium (OGC), the Apache Software Foundation (ASF), and the Open Source Geospatial Foundation (OSGeo) invite software developers to a joint Virtual Code Sprint from the 8th to 10th of March 2022. More information to follow.
