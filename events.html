Events

  • 2021-02-17

    Joint OGC OSGeo ASF Code Sprint (Virtual)

    The Apache Software Foundation (ASF), Open Geospatial Consortium (OGC), and Open Source Geospatial Foundation (OSGeo) are currently planning a joint Virtual Code Sprint to be held during the last week of February 2021. The code sprint will cover multiple ASF and OSGeo projects, and OGC standards. This Sprint is open to all participants in ASF, OSGeo and OGC, even if you are not active members or contributors to the efforts of the organizers.

  • 2020-09-29

    OGC API - Common and OGC API - Features Sprint (Virtual)

    The Open Geospatial Consortium (OGC) invites interested developers to the OGC API - Common and OGC API - Features Sprint, to be held through remote participation/web-conferencing from 29th to 30th of September 2020. The sprint will cover the draft OGC API - Common standard and the draft Part 4 of the OGC API - Features standard.
