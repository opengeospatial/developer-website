    July 2021 OGC API Virtual Code Sprint

    The OGC invites software developers to the July 2021 OGC API Virtual Code Sprint, to be held from July 21st to July 23rd, 2021. The code sprint will cover the draft OGC API – Processes, OGC API – Records, and OGC API – Coverages specifications.

    The OGC invites software developers to the May 2021 OGC API Virtual Code Sprint, to be held from May 26th to May 28th, 2021. The code sprint will cover the draft OGC API – Maps, OGC API – Tiles, and OGC API – Styles specifications.

  • 2021-02-17

    Joint OGC OSGeo ASF Code Sprint (Virtual)

    The Apache Software Foundation (ASF), Open Geospatial Consortium (OGC), and Open Source Geospatial Foundation (OSGeo) held a joint Virtual Code Sprint during the last week of February 2021. The code sprint covered multiple ASF and OSGeo projects, and OGC standards. This Sprint was open to all developers.
