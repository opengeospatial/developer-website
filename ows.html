OGC Web Services

OGC Web Services were created more than ten years ago, when the web technologies looked very different from what is used nowadays. OGC encourages the transition from OWS to OGC API Standards and there is a deprecation plan in place for OWS standards. If you are going to implement a standard from scratch, please consider using one of the OGC APIs instead.
