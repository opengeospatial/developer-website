OGC Web Services

OGC Web Services were created more than ten years ago, when the web technologies looked very different from what is used nowadays. OGC encourages the transition from OWS and "Technical Committee Policies and Procedures 05-020r29" presents a process for deprecating API elements in OGC Standards. If you are going to implement a standard from scratch, please consider using one of the OGC APIs instead.
